# Spotify-Clone

# 🎵 Spotify Clone - Music Player Web App

A fully functional **Spotify Clone Music Player** built using modern web technologies.
Users can browse songs, play music, and enjoy a smooth music streaming experience.

🌐 **Live Demo:**
👉 https://rohanmusicplayer.vercel.app/

---

# 🚀 Features

* 🎧 Play songs instantly
* ⏯️ Pause / Resume music
* ⏭️ Next & Previous song controls
* 📃 Display song list dynamically
* 🎵 Smooth audio playback
* 📱 Responsive design
* 🌐 Backend API integration
* ⚡ Fast loading UI

---

# 🎮 How to Use

1. Open the website
2. Click **Get Songs** button
3. Select any song from the list
4. Use controls to:

   * Play
   * Pause
   * Next
   * Previous

Enjoy music 🎵

# 🧑‍💻 Installation (Run Locally)

## Clone Repository

```
git clone https://github.com/your-username/Spotify-Clone.git
```

## Go to project folder

```
cd Spotify-Clone
```

## Install dependencies

### Backend

```
cd backend
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm run dev
```
